- The software supports BrainTree or Stripe very well.
 	- Its a free of charge tool that the founder created because of the frustration he felt helping fortune 500 CEOs measure their MRR (monthly recurring revenue)

 	free platform
 	- setup in a few minutes

 	- Monetization Model: B2B content inside sales

 	Everyone hates marketers

 	- Advice on packaging

 	Stop giving unlimited free - makes it hard to upgrade these users
 	- educate users about what each feature means

 	flexible scheduling
 	- price data sources
 	- web hooks

 	- Always focus on why someone is using a specific tool - personas

 	e-Commerce
 	- Real Estates
 	- Traders

 	- The more packages you have available the less money you leave on the table

 	so that users don't think this software is not suitable for them
 	- Packaging targeted at specific persona helps the target audience to know immediately which package he should go for. Don't force them to think too much

 	- 5 is a good number

 	- Advice on running inside sales process

 	spend time to study customer's website so as to use the same words when communicating with them
 	- seek to offer advice based on market knowledge
 	- versus giving official presentation
 	- offered to send customers his way - referral

 	- Very nicely setup inside sales process
